Update data bulanan

Sunting data di bawah, lalu klik Unduh data.js. Ganti file js/data.js pada folder project dengan file hasil unduhan — profil akan menampilkan data baru tanpa perlu menyentuh kode.

Metadata dipakai untuk judul dokumen, badge «DATA …», «MONITORING …», dan footer pada halaman profil.

Perbarui data dengan mengunggah file — pilih dulu bagian yang mau diperbarui. Format: data.js / JSON (hasil unduhan halaman ini) untuk semua bagian, atau Excel (.xlsx) / CSV untuk Kinerja/Pelanggan/Sistem. Unduh template Excel di bawah — sudah terisi data saat ini, tinggal disunting lalu diunggah kembali.
